Web Notes

A little something about Web notes:

Web notes is a simple and rogressive notes Web App.

This app serves the following features:

  • It helps you write notes which are then saved to your browser's local storage.
  • You can change the view between list view and grid view.
  • You get to copy the entire note with just one click on the copy icon.
  • You can bold, italisize and underline parts of your note for better comprehension.
  • It is light-weight and thus loads almost instantly.
  • It's open-source!


If you liked using this app, consider a small donation so that I keep working on improving it further and the experience remains ad-free. Icons are taken from Flaticon.
